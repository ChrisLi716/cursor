## 💡 学习建议

### ✅ 推荐做法

1. **先实践后理论**
   - 先看示例，动手尝试
   - 再深入了解原理
   - 这样理解更深刻

2. **边学边用**
   - 看完一个场景就立即尝试
   - 在实践中理解工具调用

3. **选择性深入**
   - 根据需求选择学习深度
   - 不需要全部掌握才能使用

4. **保存为参考**
   - 将文档保存，需要时查阅
   - 特别是工具调用格式

### ❌ 避免的做法

1. **不要只看不做**
   - 必须动手实践
   - 只有实际使用才能理解

2. **不要一开始就深入技术细节**
   - 先掌握基本使用
   - 需要时再深入了解

3. **不要跳过快速开始**
   - 快速开始章节很重要
   - 能让你立即看到效果
