## 💡 常见问题

### Q: 我需要配置什么才能使用 MCP 工具？
**A:** 不需要！Cursor 内置的浏览器扩展 MCP 工具已经可以直接使用。只有需要外部服务（如文件系统、GitHub）时才需要配置。

### Q: 如何在 Cursor 中使用 MCP 工具？
**A:** 直接在聊天中描述你的需求即可，AI 会自动调用合适的 MCP 工具。无需手动调用。

### Q: 我能看到工具调用的过程吗？
**A:** 在 Agent 模式下，可以看到 AI 的工具调用过程。不过通常你只需要关注最终结果。

### Q: 如何验证 MCP 是否工作？
**A:** 最简单的方式是在 Cursor 中请求一个需要浏览器工具的操作，例如"访问某个网页并截图"。
